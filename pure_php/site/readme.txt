# Описание работы и установки приложения

* Дамп базы данных находится в `db/DataBase.sql`
* База данных MYSQL. 
* Папки `lib`, `view_parts` должны быть защещены от чтение и изменения из вне.
* `PHP 5.4>` но лучше `PHP7>`.
* Настройки БД `lib/main/config.php`
* Основные управляюущие классы находятся в папке `lib/main`. `App` и `AdminApp`
